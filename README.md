# ForkIt — documentation

Public specifications and planning for **ForkIt** (“I’ll cook it myself”): a DoorDash-inspired flow for discovering recipes, planning meals, and building grocery lists without delivery fees.

Application source is maintained in a **separate private repository** (`forkit-source` in the same local layout as this repo).

## What lives here

|------|---------|
| [`SPEC.md`](./SPEC.md) | Product requirements and UX behavior |
| [`PLAN.md`](./PLAN.md) | Milestones and progress checklist |

| [`ESR.md`](./ESR.md) | Smart-question habits for support and bugs |
| [`AGENTS.md`](./AGENTS.md) | AI / contributor workflow for this docs repo |
| [`GEMINI.md`](./GEMINI.md) | Gemini-specific project notes |
